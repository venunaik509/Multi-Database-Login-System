# Signup/Login Authentication System

This project is a simple web app for signup/login authentication using HTML, CSS, JavaScript, Firebase Authentication, and Firestore.

## Features
- Signup form collects Full Name, Email, Password, and Phone Number
- Email and Password are stored in Firebase Authentication
- Full Name and Phone Number are stored in Firestore linked to the user's UID
- Login form requires Email and Password
- After login, displays a welcome message with the user's full name from Firestore and a logout button
- Simple responsive CSS for both forms
- Firebase initialization code with placeholders for API keys and config values

## Setup
1. Replace the Firebase config placeholders in `firebase.js` with your actual Firebase project configuration.
2. Open `index.html` in your browser to use the app.
