# Login/Signup Website with Firebase

## Features
- Signup and login with email and password
- User data stored in Firebase (Firestore or Realtime Database)
- Secure authentication using Firebase Auth
- Simple, user-friendly frontend (HTML/CSS/JS)

## Structure
- `backend/`: Node.js + Express server, connects to Firebase
- `frontend/`: Static HTML/CSS/JS for UI

## Setup
1. Configure Firebase project and service account (see backend instructions)
2. Run backend: `npm install` then `npm start` in the backend folder
3. Open `frontend/index.html` in your browser
